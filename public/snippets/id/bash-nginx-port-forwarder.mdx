Script ini digunakan untuk melakukan konfigurasi server NGINX.

```bash
server {
        listen 80;
        server_name serv.block.here; // eg; blabla.malahngoding.com

        location / {
                proxy_pass http://127.0.0.1:3500; // eg; http://127.0.0.1:3000
                proxy_set_header X-Forwarded-For $remote_addr;
                proxy_set_header Host $http_host;
        }
}
```
