Melakukan git update pada semua repository di dalam folder yang kamu buat. Sekali run akan melakukan `git pull` pada setiap folder git kamu.

```bash
#! /usr/bin/env bash
for dir in ./*/
do
  cd ${dir}
  git status >/dev/null 2>&1
  # check if exit status of above was 0,
  # indicating we're in a git repo
  [ $(echo $?) -eq 0 ] && echo "Updating ${dir%*/}..." && git pull
  cd ..
done
```
